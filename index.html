Project Three: fullTank

An API-fueled petrol-price tracker





Burn rubber, not money.

For our group project, we were told to create a Django app, and in a time of volatile petrol prices, we decided to build something useful for the community. For the project, the information had to be user-submittable, which we recognize isn't the ideal form for a production app. We therefore sourced a paid-for gas price API, which we have waiting in the wings, should we ever take the project further. It is also optimized for mobile.
